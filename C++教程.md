# 结构体

C/C++ 数组允许定义可存储相同类型数据项的变量，但是`结构体`是 C++ 中另一种用户自定义的可用的数据类型，它允许您存储不同类型的数据项。

## 结构体的定义

```c++
struct type_name 
{ 
    member_type1 member_name1; 
    member_type2 member_name2; 
    member_type3 member_name3; 
};
```
